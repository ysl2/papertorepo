# Paper Detail Quick Links Trial Plan

## Context

The paper detail drawer already contains arXiv and Repository links, but they appear below the abstract and metadata. This trial adds quick jump links near the existing status/category/date chips so the visual placement can be evaluated before committing to the final interaction design.

## Plan

- Add `arXiv` quick link to the paper detail drawer tag row.
- Add `GitHub` quick link to the same row when `primary_github_url` exists.
- Keep table interactions unchanged.
- Do not change backend schema, API, or repo correction behavior.
- Run frontend build after the UI change.

## Scope

- Frontend-only, minimal visual trial.

# Language Blank Filter Option

## Requirement

Add an explicit `(Blank)` option to the frontend Papers table `Language` column value filter so users can filter rows whose repository language is empty.

## Plan

1. Extend the compact set filter with an opt-in blank option label and sentinel value.
2. Enable that blank option only for the Papers table `Language` column.
3. Verify that blank rows match only when `(Blank)` is selected and existing non-empty language options continue to work.
4. Run frontend build/lint checks.

## Success Criteria

- The `Language` column filter shows `(Blank)` when any visible row has an empty language.
- Selecting `(Blank)` includes rows with empty language values.
- Non-empty language options keep their current labels and filtering behavior.

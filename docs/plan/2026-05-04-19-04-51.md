# Adjust Toolbar Refresh and Reset Order

## 背景

当前表格 toolbar 中 `Reset view` 位于 `Refresh` 前面，且文案较长，会占用搜索框左侧的横向空间。用户希望顺序改为 `Refresh Reset`，并将 `Reset view` 缩短为 `Reset`。

## 计划

1. 只修改 `frontend/src/components/AgGridSheet.tsx`，不改 refresh、reset、搜索、SQL 或 rows summary 的行为。
2. 将 `toolbarActions` 渲染位置移动到 reset 按钮前面，使最终顺序为 `Columns`、`Freeze`、`Refresh`、`Reset`、搜索框。
3. 将 reset 按钮可见文案从 `Reset view` 改为 `Reset`。
4. 给 reset 按钮保留 `title="Reset view"` 和 `aria-label="Reset view"`，确保完整语义和 hover 提示仍然存在。
5. 运行 `npm --prefix frontend run build` 验证前端构建通过。

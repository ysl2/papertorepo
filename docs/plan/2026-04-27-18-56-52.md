# Papers table category order and header width plan

## Summary

Adjust the frontend Papers table column configuration so `Category` appears before `Title`, and widen `Published` and `Language` so their headers display more reliably.

## Implementation

- Move the `Category` column to the left of `Title`.
- Increase the preferred widths for `Published` and `Language`.
- Bump the Papers grid persistence id to avoid stale saved column order and widths.

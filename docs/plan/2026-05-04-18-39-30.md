# Compact Search/SQL Toggle

## 背景

当前前端搜索框内的 `Search | SQL` 切换器视觉样式可接受，但宽度偏大。在右侧 refresh papers 进度条出现时，搜索栏可用宽度会被进一步压缩，因此需要让切换器更紧凑，同时不改变 SQL/Search 的行为逻辑。

## 计划

1. 只修改 `frontend/src/App.css`，不改 React 状态、SQL 执行逻辑、API 或环境变量行为。
2. 将搜索框内切换器宽度从 `112px` 缩小到 `88px`。
3. 将 SQL 开启时输入框左侧 padding 从 `128px` 调整到 `104px`，避免文字与切换器重叠。
4. 将切换器按钮横向 padding 从 `0 6px` 调整到 `0 3px`，并把字体从 `11px` 调整到 `10px`，保留完整 `Search` 和 `SQL` 文案。
5. 运行 `npm --prefix frontend run build` 验证前端构建通过。
